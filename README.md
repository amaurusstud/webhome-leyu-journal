# webhome-leyu-journal

本仓库用于归档和发布多个独立的 HTML 页面，主要面向个人项目或日常记录用途。仓库中的页面以静态 HTML 文件形式存放，供浏览、参考或直接部署。

## 项目简介

这是一个以 HTML 页面为主要内容的归档仓库。每个页面独立存在，不依赖特定的后端服务或数据库。仓库结构简单，适合存放各类无需复杂交互的静态内容，例如个人笔记、信息展示页、工具页面等。

## 目录说明

仓库根目录下存放各 HTML 文件，可根据需要创建子目录进行分类。建议遵循以下结构：

- `index.html` — 主页或导航页（如有）
- `pages/` — 存放主要 HTML 页面
- `assets/` — 存放图片、样式表、脚本等静态资源
- `archive/` — 存放较旧或不再更新的页面

具体目录结构以实际提交为准，仓库维护者可根据内容变化进行调整。

## 页面归档说明

每个 HTML 页面均独立归档，不与其他页面共享数据。页面内容可能包含文本、图片、样式或简单脚本。归档时建议：

- 页面文件命名清晰，反映内容主题
- 内部链接尽量使用相对路径
- 资源文件统一放置于 `assets/` 目录下

本仓库不保证所有页面均长期可访问，建议在本地保留备份。
